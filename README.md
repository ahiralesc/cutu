Cluster Usage Trace utilities (CUT-Utilities)


CUT utilities have been design to extract, process, transform, and analyze job and task execution traces, currently CUT-U supports:
-   GCUT, [Google Cluster Usage Traces](https://github.com/google/cluster-data)
